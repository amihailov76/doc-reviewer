# pymorphy3: морфологический анализ в детекторе инструкций

## Почему pymorphy3, а не pymorphy2

Оригинальная библиотека `pymorphy2==0.9.1` использует `inspect.getargspec()`, которая была удалена в Python 3.11. При запуске возникает ошибка:

```
AttributeError: module 'inspect' has no attribute 'getargspec'
```

`pymorphy3==2.0.2` — совместимый форк с поддержкой Python 3.11+. API библиотек идентично, поэтому в коде используется алиас:

```python
import pymorphy3 as pymorphy2
```

Это позволяет при необходимости вернуться на оригинальный pymorphy2 (если появится совместимая версия) заменой одной строки импорта.

## Почему морфологический анализ, а не список суффиксов

Детектор инструкций работает с большим объёмом разделов. Суффиксный подход (проверка окончаний `-ение`, `-ация` и т.п.) даёт ложные срабатывания на словах, которые оканчиваются похоже, но не являются девербативами. pymorphy3 выполняет полный морфологический разбор и возвращает часть речи с оценкой уверенности, что существенно снижает количество ошибок.

## Как работает определение отглагольных существительных

Функция `has_deverbal_noun_in_title()` в `backend/services/detector.py`:

1. Извлекает из заголовка все русские слова длиннее 4 символов
2. Для каждого слова вызывает `_is_deverbal_noun()`
3. Возвращает `True` если хотя бы одно слово является девербативом

Функция `_is_deverbal_noun()`:

1. Быстрая проверка суффикса — если слово не оканчивается ни на один из продуктивных суффиксов девербативов, сразу возвращает `False` (без морфоанализа)
2. Морфологический анализ через pymorphy3 — ищет разбор с тегом `NOUN` и уверенностью > 0.1

Продуктивные суффиксы девербативов (неполный список):
`-ирование`, `-ование`, `-ация`, `-ение`, `-ание`, `-тие`, `-овка` и их словоформы.

## Кеширование

Анализатор создаётся один раз при импорте модуля:

```python
_morph = pymorphy2.MorphAnalyzer()
```

Загрузка словарей занимает ~0.5 секунды при первом обращении. При последующих запусках словари берутся из кеша операционной системы.

Результаты разбора отдельных слов кешируются через `@lru_cache(maxsize=4096)` — повторный анализ одного и того же слова не выполняется.

## Первая установка

```
py -3.11 -m pip install pymorphy3==2.0.2
```

Словари русского языка входят в состав пакета (~20 МБ) и устанавливаются автоматически.
